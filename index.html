AceBerry

AceStream P2P en tu Raspberry Pi

Stack completo con proxy inteligente, interfaz web y acceso remoto vía Tailscale. Un solo comando para instalarlo todo.

Instalación

$ curl -fsSL https://raw.githubusercontent.com/AlexAdiaconitei/aceberry/main/proxy-setup/install.sh | sudo bash

Ejecutar en la Raspberry Pi · Requiere sudo

Ver en GitHub Cómo funciona

Requisitos

🍓

Raspberry Pi

RPi 4 o superior recomendado. Compatible con ARM64, ARMv7 y AMD64.

🐋

Docker

El instalador configura Docker automáticamente si no está presente en el sistema.

🔒

Tailscale

Acceso VPN seguro desde cualquier dispositivo. Cuenta gratuita suficiente.

/content_id/…/stream.ts

Cómo funciona

En tres pasos desde cero hasta streaming

1

Instalar

Ejecuta el one-liner en la Raspberry Pi. El script instala Docker, Tailscale, descarga las imágenes y arranca el stack automáticamente.

2

Conectar

Une la Raspberry Pi a tu red Tailscale con tailscale up. Accede a la interfaz web desde cualquier dispositivo con Tailscale instalado.

3

Reproducir stream

Abre la interfaz en http://<pi-ip>:8080, pega un ID de AceStream y obtén la URL para VLC o tu reproductor favorito.

CLI acestream-ctl

Gestiona el stack desde la terminal

raspberry@pi:~
$ acestream-ctl status

Ver estado de todos los contenedores

$ acestream-ctl url <ACESTREAM_ID>

Obtener URL de stream para un ID

$ acestream-ctl logs

Ver logs de todos los servicios

$ acestream-ctl restart
